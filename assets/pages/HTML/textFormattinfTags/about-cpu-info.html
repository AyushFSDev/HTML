CPU Information

Definition of CPU

A CPU is a computer's central processing unit. It is a circuit board that performs the basic functions of a computer, such as arithmetic, logic, and input/output (I/O). The CPU is the heart of a computer. It is responsible for the processing of instructions and data, and for the execution of programs.

What is Primary Memory?

Primary memory is the memory that stores the data that is being processed by the CPU. It is also known as the "working memory".
Primary memory, also known as main memory or RAM (Random Access Memory).

What is Secondary Memory?

Secondary memory is the memory that stores the instructions that the CPU executes. It is also known as the "program memory".
Secondary memory, also known as secondary memory or ROM (Read Only Memory).

What is Processor?

The processor is the brain of the computer. It is responsible for performing the basic functions of a computer, such as arithmetic, logic, and input/output (I/O).
